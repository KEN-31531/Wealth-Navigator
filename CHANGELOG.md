# 版本更新紀錄

## v1.2.0 - 2026-01-28

### 新增功能

- 用戶註冊流程：加入好友時需輸入姓名與匯款後五碼
- Google Sheets CRM 整合：自動儲存用戶資料與測試結果
- 註冊狀態持久化：即使伺服器重啟，註冊進度不會遺失
- 測試結果頁新增「領取三招抗通膨秘笈」PDF 連結按鈕

### 介面調整

- 品牌色系更新：選中項目 #FFE153（黃色）、按鈕 #408080（青綠色）
- 註冊完成訊息改為「導航上線」
- 多選題必須至少選擇一個選項才能繼續

### 技術改進

- 支援從環境變數 `GOOGLE_CREDENTIALS` 讀取憑證
- 新增 `google_sheets.py` 模組
- 新增 `user_registration.py` 模組

## v1.1.0 - 2026-01-27

### 功能更新

- 多選題 toggle 功能：點擊選項可切換選取狀態
- 多選題視覺回饋：已選項目顯示黃色背景與打勾

### UI 修正

- 改用 Flex Message 顯示按鈕（白色背景、垂直排列）
- 修復 Flex Message spacer 錯誤
